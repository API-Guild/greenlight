[MDX](https://mdxjs.com/getting-started) is a superset of [Markdown](https://www.markdownguide.org/getting-started/). 
This is the technology that allows embedding [Tableau](https://www.tableau.com/embedded-analytics) dashboards unto 
Markdown blog posts. The [Greenlight](https://api-guild.github.io/greenlight/about) team has provided an additional 
set of reusable components to help you write engaging content for your blog.

In MDX, components are written with [JSX](https://en.wikipedia.org/wiki/JSX_(JavaScript)) - the language used by the 
[React](https://reactjs.org/docs/introducing-jsx.html) JavaScript library. This document describes the syntax for each 
component and provides examples so you can use them for your own content.

## Title

```javascript
<Title 
    title="Awesome Section"
    titleSize={1}
    titleColor="primary"
    subtitle={<div>Use this component to replace <strong>headings</strong></div>}
    subtitleSize={3}
    subtitleColor="grey-light"
/>
```
### Result:
<Title 
    title="Awesome Section"
    titleSize={1}
    titleColor="primary"
    subtitle={<div>Use this component to replace <strong>headings</strong></div>}
    subtitleSize={3}
    subtitleColor="grey-light"
/>

## Callouts

```javascript
<Callout
    type="primary"
    title="Callout"
    subtitle={

    </div>
    }
```

## Cards
```javascript
<Card
    header="Sample Card Component"
    image="https://bulma.io/images/placeholders/1280x960.png"
    content={<span>Lorem ipsum dolor sit amet, consectetur adipiscing elit. 
            Phasellus nec iaculis mauris.
            <a href="/"> @bulmaio</a><a href="/"> #css </a><a href="/">
            #responsive</a></span>}
    footer={[
        <a href="/" className="card-footer-item">Save</a>, 
        <a href="/" className="card-footer-item">Edit</a>, 
        <a href="/" className="card-footer-item">Delete</a>,
    ]}
/>
```
